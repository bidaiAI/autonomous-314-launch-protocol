## What this repository is

Autonomous 314 is an open-source, creator-first, EVM launch protocol:

- pre-graduation trading lives inside the launch contract itself
- graduation hands liquidity to a canonical V2-style DEX
- the protocol is designed to be platform-independent
- the official frontend is a reference implementation, not the gatekeeper
